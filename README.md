This repository serves as a professional portfolio of data engineering and analytics projects completed during the MSDA program at Western Governors University.

## 🚀 Featured Projects

### D597 — Data Management (SQL & NoSQL)
* **[EcoMart Sales Analysis & Relational Design](<D597 - EcoMart SQL Database/README.md>)**
  * Engineered a scalable 2NF PostgreSQL database from flat files, identifying that discontinuing the 'Fruit' category could mitigate significant losses due to low profitability and high perishability.
* **[Product Partnership Analysis (NoSQL)](<D597 - NoSQL MongoDB Database/README.md>)**
  * Architected a MongoDB solution that identified Garmin and Apple as high-profit partners while uncovering a critical data-collection gap regarding premium product reviews.

### D598 — Statistics & Specialized Analysis
* **[The Compounding Effect: Smoking & BMI (Python)](<D598 - How Smoking Amplifies the Financial Risk of High BMI/README.md>)**
  * Quantified a $27,110 median charge increase for smokers and utilized Spearman correlation to prove that BMI's financial impact is almost exclusively limited to the smoking demographic.
* **[Financial Risk & Portfolio Performance (R)](<D598 - Investment Analysis in R/README.md>)**
  * Built an automated R-based analytical engine to isolate insolvent clients with negative debt-to-equity ratios and benchmark regional revenue performance.

### D599 — Data Mining
* **[Driving Retail Strategy (Market Basket Analysis)](<D599 - Market Basket Analysis in Python/README.md>)**
  * Leveraged the Apriori algorithm to identify high-lift product bundles and statistically proved that high-priority shipping is the primary driver of satisfaction for corporate segments.

### D600 — Regression & PCA
* **[Real Estate Price Prediction & Luxury Classification (Linear/Logistic Regression)](<D600 - Price Prediction & Luxury Classification for Real Estate/README.md>)**
  * Developed a dual-model suite using Linear and Logistic regression to predict housing prices ($R^2=0.67$) and classify luxury assets with 89% accuracy.

### D602 — Deployment & MLOps
* **[Airline Delay Prediction: End-to-End ML Lifecycle & API Deployment](<D602 - Airline Delay Prediction%3A End-to-End ML Lifecycle %26 API Deployment/README.md>)**
  * Architected a complete MLOps pipeline using MLFlow and DVC to tune and track flight delay models, deployed via a Dockerized FastAPI service with full unit testing.

### D603 — Machine Learning
* **[Hospital Readmission Prediction using Gradient Boosting](<D603 - Hospital Readmission Prediction using Gradient Boosting/README.md>)**
  * Engineered a Gradient Boosting model with 98% accuracy to identify 'Initial_days' as the primary driver of readmissions, using SHAP for advanced model interpretability.

### D607 — Cloud Databases
* **[Cloud Database Architecture & Migration to GCP Firestore](<D607- Cloud Data base Architecture %26 Migration to GCP Firestore/README.md>)**
  * Designed and implemented a scalable GCP Firestore (NoSQL) migration strategy for a high-volume retailer, handling 10 TB of daily transaction data with real-time query support.

### D608 — Data Processing & Pipelines
* **[Data Pipelines with Apache Airflow: Automated Music Streaming ETL](<D608 - Data Pipelines with Apache Airflow%3A Automated Music Streaming ETL/README.md>)**
  * Developed a production-ready Airflow DAG with custom operators to automate the staging and transformation of music data from S3 into Redshift with integrated quality checks.

### D609 — Data Analytics at Scale
* **[STEDI Human Balance Analytics: Data Lakehouse Architecture](<D609 - Data Lakehouse Architecture/README.md>)**
  * Architected an AWS-based Lakehouse (S3, Glue, Athena) to process high-volume IoT sensor data, implementing a Spark-driven Medallion architecture with automated privacy/consent filtering.
